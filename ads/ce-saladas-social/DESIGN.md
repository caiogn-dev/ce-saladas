# Cê Saladas Design Reference

Fonte oficial de planejamento: [docs/video-ad/DESIGN.md](/home/graco/WORK/ce-saladas/docs/video-ad/DESIGN.md)

## Core Identity

- Warm base: `#fdfbf7`
- Leaf green: `#649e20`
- Terra accent: `#f97316`
- Text strong: `#36332f`
- Optional dark canvas: `#16100a`

## Type

- Display: `Fraunces`
- UI/body: `Inter`
- Optional accent detail: `Caveat`

## Direction

- Food first, UI second
- Organic motion, not tech-demo motion
- Premium local food brand, not marketplace aesthetic
- CTA must drive to the site/cardápio
